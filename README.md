# AI Resume Screening System with Explainability

## 📌 Objective
This project evaluates resumes against a job description using an AI-inspired pipeline.

## 🚀 Features
- Skill Extraction
- Resume Matching
- Scoring (0–100)
- Explanation Generation

## 🔄 Pipeline
Resume → Extract → Match → Score → Explain

## 📊 Test Cases
- Strong Candidate
- Average Candidate
- Weak Candidate

## 🛠 Tech Used
- Python
- Google Colab

## ⚠️ Note
This implementation uses a rule-based approach due to API limitations. 
The architecture is designed to integrate LangChain and LangSmith for tracing.
